how to contribute
Here are some suggestions if you would like to help.
  1. Have you computer start mining for muids while you consider this list.
  2. Better yet ... optimize the miner while you are at it.
  3. Start crawling with thoughtful, well designed statistical algorithms.
  4. Produce a live, clean JSON source of data of civic significance that changes at least five times an hour.
  5. Write an R client. It is a crime we don't have R powered crawlers yet.
  6. Write a Julia client.
  7. Add to the ways data can be plotted by providing plotly examples (or other ways).
  8. Make suggestions by creating a github issue.
  9. Become a contributor and then scan the list of projects.
