Crawling

It is easy to write programs that look for streams to predict and submit scenarios.

Step 1. Install microprediction

    pip install microprediction

Step 2. Get a key

As explained here, you need a write_key to submit. At minimum click on this link to create on. However we encourage you to visit muid.org to understand MUIDs and mine a better key. You can run the mining bash script. Using a difficult key may greatly increase your crawler's longevity. See the table of bankruptcy levels in the microprediction package's README.

Step 3. Run the default crawler

    from microprediction import MicroCrawler
    crawler = MicroCrawler(write_key="YOUR_WRITE_KEY_HERE")
    crawler.run()

Step 4. Examine performance

   mw.get_performance()
   mw.get_overall()
   mw.get_home()
   mw.get_errors()
   mw.get_transactions()
