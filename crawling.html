Crawling
It is easy to write programs that look for streams to predict and submit scenarios.
[Step 0] Install microprediction
    pip install microprediction
Or see microprediction from PyPI.
[Step 1] Get a key
See muid instructions. Note especially the table of bankruptcy levels in the microprediction package's README. At minimum, visit muid.org to create a minimal strength MUID for prediction submission.
[Step 2] Run the default crawler
    from microprediction import MicroCrawler
    crawler = MicroCrawler(write_key="YOUR_WRITE_KEY_HERE")
    crawler.run()
[Step 3] Examine performance
   mw.get_performance()
   mw.get_overall()
   mw.get_home()
   mw.get_errors()
   mw.get_transactions()
[Step 4, 5, ...] Improve it
This we leave in your capable hands.
