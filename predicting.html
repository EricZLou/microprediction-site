Making a Prediction
It is easy to write programs that make predictions.
[Step 0] Install microprediction
    pip install microprediction
Or see microprediction from PyPI.
[Step 1] Get a key
See muid instructions. Note especially the table of bankruptcy levels in the microprediction package's README. At minimum, visit muid.org to create a minimal strength MUID for prediction submission.
[Step 2] Choose a live data stream
Stream listing. Examining individual summaries such as /live/summary::cop.json or value histories live/lagged::cop.json or live/lagged_values::cop.json. See also plotting instructions.
[Step 3] Use the MicroWriter to submit scenarios
    from microprediction import MicroWriter
    mw = MicroWriter(write_key="744fd0bec96112c9c5ce92ee904e1a4e")  # Substitute in your own key
    scenarios = [ i*0.001 for i in range(mw.num_predictions) ]
    mw.submit(name='cop.json',values=scenarios)
[Step 4] Examine performance
    mw.get_performance()['cop.json']
    mw.get_leaderboard(name='cop.json')
    mw.get_overall()
    mw.get_home()
    mw.get_errors()
    mw.get_transactions()
[Step 5, 6, ...] Improve it
This we leave in your capable hands. Here is a start:
    mw.get_lagged_values(name='cop.json')
    mw.get_summary(name='cop.json')
Submission API
You can also make a PUT request to submit/STREAM_NAME, with values in the payload as a comma separated list. See the main API page.
